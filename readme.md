# ZDoom dev build recreation
intended date: December 31st, 1998

## Requirements
 * [MIDAS 1.1.2 static library](https://web.archive.org/web/19980520064706/http://s2.org/midas/midas112.zip) -- a hack from NTDOOM will crash with DLL.
 * [PTC 0.72 source code](https://sourceforge.net/projects/openptc/files/OpenPTC/Original%20PTC/PTC-0.73.zip/download) -- this version of ZDoom will not work with PTC unmodified. Instructions to patch are in the "Environment setup" section. We're currently using 0.73 as it seems that 0.72's source is lost.
 * [NASM](https://web.archive.org/web/19980128023126/http://s2.org/midas/midas111.zip) (optional) -- If you don't want to use NASM, #define NOASM while compiling ZDoom.
 * Microsoft Visual C++ 5.x
